If you have LeapMotion laying around, it's a good moment to connect it and give it a try!

If you don't have it, you can still move the polygons around using your mouse or finger on a touch device, but it's not half as fun! :)

Technology: Canvas, Leap Motion JS API

You need a browser that supports the canvas element to run this experiment. Chrome, Firefox, Safari or IE 9 and up will do.
